# İlm-i Sîmâ — iPhone Kurulumu

Bu klasördeki TÜM dosyaları (alt klasör yok, hepsi tek seviyede) GitHub'a
yükleyip Render'a dağıtacaksın. Detaylı adımlar için Claude'a sor.

Özet:
1. console.anthropic.com → API anahtarı al (sk-ant-...) + bakiye yükle
2. github.com → yeni repo aç → bu dosyaların hepsini yükle
3. render.com → GitHub ile bağla → Free plan
4. Render'da ANTHROPIC_API_KEY gizli değişkenini ekle
5. Verilen adresi iPhone Safari'de aç → Paylaş → Ana Ekrana Ekle

İlm-i sîmâ kültürel/tarihî bir gelenektir; bu uygulama eğlence amaçlıdır.
